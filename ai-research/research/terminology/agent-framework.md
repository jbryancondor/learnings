## Summary
Agent Frameworks are software libraries that enable creation, orchestration, and management of AI agents. Major frameworks include LangChain (modular orchestrator), AutoGen (multi-agent conversation), and CrewAI (role-based teams). Each framework emphasizes different paradigms: LangChain chains operations, AutoGen structures conversations, CrewAI simulates team dynamics. In 2025, frameworks shifted from general-purpose chains to specialized agentic patterns.

## Key Concepts
- **LangChain**: Modular orchestrator for complex, multi-step workflows
- **AutoGen**: Multi-agent conversation-first framework with structured dialogues
- **CrewAI**: Role-based task execution, team-oriented design
- **Tool Integration**: Standardized mechanism for agents to use external tools
- **Memory Management**: Built-in patterns for agent context and history
- **Task Routing**: Dynamic assignment of work to specialized agents
- **Error Handling**: Recovery patterns for agent failures
- **Customization**: Frameworks support domain-specific agent patterns
